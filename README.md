# 🏠 House Market Prediction

A Machine Learning project developed using Python to predict house prices based on features like area, number of bedrooms, bathrooms, parking, and furnishing status.

## 📌 Project Overview

This project uses regression algorithms to analyze housing data and predict market prices accurately. The main goal is to understand data preprocessing, visualization, model training, and predictive analytics using Machine Learning.

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* House Price Prediction
* Model Evaluation
* Data Visualization

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## 🤖 Machine Learning Models

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

## 📂 Project Workflow

1. Import Dataset
2. Data Cleaning
3. Data Visualization
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Prediction & Evaluation

## 📊 Dataset Features

* Area
* Bedrooms
* Bathrooms
* Parking
* Furnishing Status
* Price

## 🎯 Results

The model predicts house prices based on user input features with good accuracy and demonstrates the practical implementation of Machine Learning in real estate analytics.
